FAQ

  • Can I install Mishka-Linux permanently?
    Yes, run the installer from the live session.
  • Does it include proprietary drivers?
    No, only open-source drivers are included.
  • How do I update?
    Run sudo apt update && sudo apt upgrade.
  • How do I install apps?
    Run sudo apt install (app).
